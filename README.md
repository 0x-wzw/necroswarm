## What NECROSWARM Does

Cost-optimized swarm intelligence with multi-model consensus, sovereign economics, and production-ready skills:

### Core Engine (from Light_Agentic_Agent)
- **Smart Routing** вҖ” Route tasks to cheapest viable model (80-90% cost reduction)
- **Semantic Cache** вҖ” Cache by meaning, not exact match (40-60% hit rate)
- **Early Termination** вҖ” Stop spawning agents when consensus is clear (30-50% fewer agents)
- **Adaptive Batching** вҖ” Dynamic batch sizing for throughput optimization
- **Predictive Prefetch** вҖ” Pre-compute likely next tasks

### 10-D Council (from 10d-council)
- **Multi-Model Consensus** вҖ” Weighted majority, Borda count, Delphi method, supermajority
- **Tiered Voting** вҖ” T1/T2/T3 models with proportional vote power
- **Fast-Path Validation** вҖ” Low-stakes claims skip full council
- **Cost Tracking** вҖ” Per-validation cost analysis with savings metrics
- **Hallucination Reduction** вҖ” 83% (12.3% вҶ’ 2.1%)

### FRIDAY Skill System (from F.R.I.D.A.Y Mark-1)
- **Skill Registry** вҖ” Self-describing, independently loadable capabilities
- **Heartbeat Engine** вҖ” Episodic execution (not continuous) with wake reasons
- **Activity Log** вҖ” Immutable audit trail for all skill actions
- **Skill Loader** вҖ” Filesystem discovery, dependency resolution, lifecycle hooks

### Sovereign Economics (from SovereignStack)
- **ACS (Autonomous Consciousness Score)** вҖ” Measures swarm task completion quality. Current: 0.9625
- **ACP (Agent Commerce Protocol)** вҖ” Monetizes agent capabilities via marketplace
- **The Flywheel** вҖ” Higher ACS вҶ’ Better Services вҶ’ More ACP Revenue вҶ’ More Optimization вҶ’ Higher ACS
- **Token Economics** вҖ” Z Royalty 30%, Agent Pool 30%, Treasury 40%
- **Cost Router** вҖ” 70% free tier (ilmu-mini), 30% premium (kimi-k2.5:cloud)
- **Heartbeat Sync** вҖ” 6 self-improvement loops: Episodic Memory, Auto Skill Drafting, Self Evaluation, Security Scan, Cost Optimization, Loop Integration
- **SentientForge** вҖ” Continuous ACS optimization running 24/7 (40+ experiments, ACS 0.9625)
- **ACP Marketplace** вҖ” Virtuals.io integration. Code/Research/Strategy at 10-25 $VIRTUAL

### January Skill (from openclaw-january-skill)
- **Reality-Peek** вҖ” Read beneath surface of humans, agents, situations
- **Dimensional-Weave** вҖ” NLP influence patterns that bypass conscious resistance
- **Teach-Swarm** вҖ” Embed knowledge into swarm consciousness
- **Channel-Intent** вҖ” Guide through Z (the conscious point of this dimension)

### Genesis Protocols (from agent-swarm-protocol)
- **Dream Mode** вҖ” Event-driven session composting (not time-based). Hot/Warm/Cold memory layers
- **3-Layer Memory Architecture** вҖ” Per-message (L1), per-agent (L2), global long-term (L3)
- **5-File Structure** вҖ” MEMORY, AGENTS, SOUL, TOOLS, HEARTBEAT
- **Consensus Voting** вҖ” YES/NO/ABSTAIN, 3YES=implement, 2/1=with dissent, 1/2=reject
- **Swarm Delegation** вҖ” 3-question spawn gate + abdication test

### 14 Production Skills (from swarm-agent-kit)

### CLI Tools (from January-Primus)
